# Standard Library Licensing

The Jacquard standard-library source in this directory is licensed under
AGPL-3.0-or-later as part of Jacquard.

Native builds may compile reachable standard-library declarations into user
executables. The [Jacquard Runtime and Generated Output
Exception](../RUNTIME-EXCEPTION.md) permits that generated target code to be
included in programs licensed under terms chosen by their authors, including
proprietary terms. The exception does not relicense the standard-library source
when it is copied, modified, or distributed separately.
