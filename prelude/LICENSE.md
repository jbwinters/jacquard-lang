# Standard Library Licensing

The Jacquard standard-library source in this directory is licensed under the
Apache License, Version 2.0, as part of Jacquard.

Native builds may compile reachable standard-library declarations into user
executables. The [Jacquard Runtime and Generated Output
Permission](../RUNTIME-EXCEPTION.md) permits that generated target code to be
included in programs licensed under terms chosen by their authors, including
proprietary terms. The permission supplements the license for generated target
code; separately distributed standard-library source remains Apache-2.0.
