# Stormglass

Stormglass is an incident-response war game. One checkout policy uses `Net` and
`Clock`; handlers reinterpret those effects as a sampled service fleet and a
simulated clock. Exact enumeration compares a naive retry policy with a
resilient policy and performs posterior root-cause analysis after a bad day.

The Warp suite includes deterministic cases and two properties that become
exhaustive proofs over all 27 service worlds.

```sh
sh demos/case-studies/stormglass/run.sh
```
