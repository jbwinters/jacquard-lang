# Jacquard Trademark Policy

The Jacquard name, project identity, logos, and related marks identify the
official Jacquard project.

The software license grants copyright permissions for the code. It does not
grant trademark rights.

## Permitted Uses

You may use the Jacquard name to:

- refer accurately to the official Jacquard project
- link to the official repository or documentation
- state that your software is built with, integrates with, or is compatible
  with Jacquard, if that statement is truthful and not misleading
- describe an unmodified copy as Jacquard

## Modified Versions And Services

If you distribute or operate a modified version, fork, hosted service, packaged
distribution, or language-compatible implementation, use a distinct name.

Acceptable phrasing:

- "forked from Jacquard"
- "compatible with Jacquard"
- "uses Jacquard"

Not acceptable without written permission:

- naming a fork, hosted service, or product "Jacquard"
- using a confusingly similar name
- implying endorsement by the Jacquard project
- using Jacquard marks in a way that suggests your version is the official one

Compatibility claims should be backed by the public conformance tests for the
version you claim to support.
